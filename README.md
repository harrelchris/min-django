# min-django

Minimally deployable Django application

## Install

Run the appropriate installation script for your machine

```shell
# bash
./scripts/install.sh

# cmd
scripts\install

# powershell
.\scripts\install
```

## Run

```shell
# bash
./scripts/run.sh

# cmd
scripts\run

# powershell
.\scripts\run
```
